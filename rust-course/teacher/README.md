# Rust 苏格拉底式学习系统（本地版）

把本目录当作“长期记忆 + 课程协议”。你在新开对话时，把 `rust-course/teacher/system.md` 的内容作为起始提示（或直接让 AI 先读它），即可开始按教材系统学习。

快速用法（你对 AI 说的话）：

- `开始上课`：按 `progress.md` 的“下一节”推进
- `今天我只学 30 分钟`：缩短一节课的深度/练习量
- `切换老师：小七/清清/阿雨`：切换授课风格
- `复盘上节课`：先做回忆与错题回收
- `我想看微信群聊`：读取 `wechat_unread.md` 并回复（然后把内容并入 `wechat_group.md`）
- `课后整理`：更新 `progress.md`、`diary.md`、`book_revision_notes.md`、`session_archive.md`

教材锚点：`rust-course/src/SUMMARY.md`
